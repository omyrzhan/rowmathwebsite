2. Область определения и область значений функции

Что такое область определения (D(f))?

Область определения — это все те значения переменной x, которые мы можем подставить в функцию, и при этом функция будет иметь смысл (не будет «ломаться»).

Представь, что функция — это машина. Ты кидаешь внутрь число x, а она выдаёт результат y = f(x). Но не каждое число можно кидать — некоторые числа машина просто «не переваривает».

Пример 1: Квадратный корень

Рассмотрим функцию f(x) = √(x − 2)

Внутри корня должно быть неотрицательное число (в обычной школьной математике корень из отрицательного не берём).

Значит: x − 2 ≥ 0 → x ≥ 2

Область определения: x ≥ 2 или в интервальной записи [2; +∞)

Попробуй сам: f(x) = √(x−2)
→ 2

Пример 2: Деление на ноль — нельзя!

f(x) = 1/(x−3)

Знаменатель не может быть равен нулю → x − 3 ≠ 0 → x ≠ 3

Область определения: все действительные числа, кроме x = 3
В интервалах: (−∞; 3) ∪ (3; +∞)

Под логарифм можно подставлять только положительные числа.

Значит: x > 0

Область определения: (0; +∞)

Пример 4: Сложная функция

f(x) = √(x+1)/(x−5)

Требования одновременно:

  • Под корнем: x + 1 ≥ 0 → x ≥ −1
  • Знаменатель ≠ 0: x ≠ 5

Область определения: [−1; 5) ∪ (5; +∞)

Область значений (E(f)) — какие ответы может выдавать функция

Это все возможные y, которые функция может «выдать» при допустимых x.

Пример: f(x) = x²

Квадрат любого действительного числа ≥ 0.

Область значений: [0; +∞)

Пример: f(x) = √(x−2)

Корень всегда ≥ 0, и может быть сколь угодно большим.

Область значений: [0; +∞)

Пример: f(x) = 1/x

Функция никогда не равна 0, может быть любой другой число (положительное и отрицательное).

Область значений: ℝ \ {0} (все действительные, кроме нуля)
