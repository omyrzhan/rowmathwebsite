4. Квадратичные функции — это параболы!

Что такое квадратичная функция?

Квадратичная функция — это функция, в которой самая высокая степень x равна 2.

y = a x² + b x + c или f(x) = a x² + b x + c

График любой квадратичной функции — это парабола.

  • a — главный герой: решает, вверх или вниз открыта парабола и насколько она узкая/широкая
  • b слегка сдвигает параболу влево-вправо
  • c сдвиг вверх-вниз (это точка пересечения с осью Y)

Поиграй с параболой в реальном времени!

Как a влияет на параболу? (самое важное!)

a > 0 → парабола открыта вверх (улыбка)

Есть минимум. Например: y = x², y = 2x², y = 0.5x²

Чем больше a — тем уже и круче парабола!

a < 0 парабола открыта вниз (грустная)

Есть максимум. Например: y = −x² (перевёрнутая чашка), y = −3x²

a = 0 это уже НЕ квадратичная, а линейная функция!

Вершина параболы — самая важная точка

У каждой параболы есть вершина — самая высокая или самая низкая точка.

Координата x вершины: xᵥ = −b/(2a) (запомни навсегда!)

Найдём вершину для y = 2x² − 8x + 5
xᵥ = −(−8)/(2·2) = 8/4 = 2
yᵥ = 2·4 − 8·2 + 5 = 8 − 16 + 5 = −3
Вершина в точке (2; −3) — это минимум

Жизненные примеры квадратичных функций

1. Бросок мяча вверх
Высота h(t) = −5t² + 20t + 1.5 (парабола вниз, максимум — высшая точка полёта)
2. Прибыль магазина
Если слишком поднять цену — покупателей станет меньше.
Прибыль P(x) = −2x² + 100x − 1000 (есть оптимальная цена!)
3. Площадь прямоугольника с фиксированным периметром
Периметр 40 м → P = 2(l + w), l + w = 20
S = l·w = l·(20 − l) = −l² + 20l максимум при l = 10 м (квадрат!)

Как найти корни (где парабола пересекает ось X)?

Два корняD > 0y = x² − 5x + 6 → D=25−24=1 → корни 2 и 3
