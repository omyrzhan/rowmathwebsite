Наивная теория множеств — 5. Упражнения и великий парадокс

Закрепляем всё, проверяем себя и… узнаём, почему «наивная» теория множеств привела к грандиозному кризису в математике 100 лет назад

Интерактивные упражнения (проверяются мгновенно)

1

Пусть A = {1, 3, 5, 7}, B = {3, 5, 7, 9}
Найдите A ∪ B, A ∩ B, A \ B и B \ A.

2

Является ли это функцией?
{ (Маша, 12), (Петя, 11), (Маша, 13) }

Нет! Маша не может одновременно иметь возраст и 12, и 13 лет.
Правило функции нарушено: один вход → два разных выхода.

4

Сколько элементов в множестве всех подмножеств множества {a, b, c}?
(Это называется булеаном или степенным множеством)

Подмножества: ∅, {a}, {b}, {c}, {a,b}, {a,c}, {b,c}, {a,b,c} → всего 8 = 2³
Правило: для n элементов → 2ⁿ подмножеств.

Великий парадокс Бертрана Рассела (1901 год)

«А существует ли множество всех множеств,
которые не содержат сами себя в качестве элемента?»

Обозначим это множество буквой R:

R = { X | X — множество и X ∉ X }

Теперь задаём коварный вопрос:

Принадлежит ли R самому себе?
R ∈ R ?

Рассуждаем:

  • Если R ∈ R → по определению R в R должно выполняться R ∉ R → противоречие
  • Если R ∉ R → по определению R в R попадают только те, кого в себе нет → значит R ∈ R → снова противоречие

В любом случае — противоречие!

Вывод Рассела: в «наивной» теории множеств (где можно писать «множество всех множеств с любым свойством») возникают парадоксы.
Поэтому математики придумали более строгую аксиоматическую теорию множеств (Цермело — Френкеля), где такие «слишком большие» множества запрещены.

Можно ли в обычной математике создать множество всех книг, в которых про эту книгу ничего не написано?
