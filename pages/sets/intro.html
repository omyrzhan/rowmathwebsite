Naive Set Theory — 1. Introduction

Basic ideas: what is a set, examples, naive membership.

What is a set?

A set is a collection of distinct objects, called elements or members. We write sets using curly braces like this: 

{red, green, blue}

Order doesn't matter and duplicates are ignored:

  • {1, 2, 2, 3} is exactly the same as {1, 2, 3}
  • {apple, banana, apple} = {apple, banana}

Notation

We use special symbols:

  • a ∈ A → "a is an element of set A" (the ∈ symbol means "is a member of")
  • a ∉ A → "a is NOT an element of A"
  • ∅ or {} → the empty set (a set with nothing inside)

More real-life examples of sets

  • The set of all planets in our solar system:
    {Mercury, Venus, Earth, Mars, Jupiter, Saturn, Uranus, Neptune}
  • The set of even digits:
    {0, 2, 4, 6, 8}
  • The set of vowels in English:
    {a, e, i, o, u}
  • The set of all integers: {..., -3, -2, -1, 0, 1, 2, 3, ...} (infinite in both directions)
  • The empty set (nothing): ∅ or {}

Quick quiz for beginners

Is 5 in the set {1, 2, 3, 4}?

No → we write 5 ∉ {1,2,3,4}

Is banana in {apple, banana, cherry}?

Yes → banana ∈ {apple, banana, cherry}
