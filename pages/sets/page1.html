Наивная теория множеств — 2. Обозначения и названия

Самые важные математические символы и как их правильно читать вслух

Как называть и записывать множества

Обычно множества обозначают большими латинскими буквами:

A = {1, 3, 5, 7, 9}          ← множество A
B = {красный, синий, зелёный}  ← множество B
C = { }                        ← пустое множество (можно также писать ∅)

Элементы множества обозначают маленькими буквами:

x ∈ A    → «икс принадлежит множеству А»
5 ∉ B    → «пятёрка НЕ принадлежит множеству B»

Главные символы, которые нужно знать

  • A = {1, 2, 3} — «Множество A равно множеству из элементов 1, 2, 3»
  • a ∈ A — «a принадлежит A» или «a является элементом A»
  • a ∉ A — «a не принадлежит A»
  • ∅ или {} — «пустое множество» (в нём вообще ничего нет)
  • A ⊆ B — «A является подмножеством B»
    Означает: каждый элемент из A есть и в B (A может быть равно B)
  • A ⊂ B — «A является строгим (собственным) подмнножеством B»
    A ⊆ B, но A ≠ B — в B есть что-то лишнее
  • A ⊇ B — «A содержит B» (то же самое, что B ⊆ A)
  • A = B — «множества A и B равны» (у них полностью одинаковые элементы)
  • |A| или card(A) — мощность (количество элементов) множества A

Интерактив: собери своё множество

Введи любые элементы через запятую. Повторы и порядок будут автоматически убраны — так работает настоящее множество!

Примеры равенства множеств (важная идея!)

Множества равны, если у них одинаковые элементы, а порядок и повторения не важны:

{1, 2, 3}     = {3, 2, 1, 1}          → да, равны!
{яблоко, груша} = {груша, яблоко, яблоко} → да, равны!
{2, 4, 6}     = {2, 2, 4, 6, 6}        → да, равны!

Примеры подмножества из жизни

Пусть B = {все ученики 10-А класса}

  • A = {Маша, Петя, Катя} → A ⊆ B (все трое учатся в 10-А)
  • C = {Ваня} → C ⊆ B (Ваня тоже в 10-А)
  • D = {Маша, Петя, Катя, Ваня, Оля} → D ⊈ B (Оля из 10-Б!

А вот пример строгого подмножества:

{1, 2} ⊂ {1, 2, 3, 4, 5}    ← строгое, потому что в правом множестве есть лишние числа

Мини-викторина для закрепления

  1. Верно ли, что {a, b} ⊆ {a, b, c}? → Да
  2. Верно ли, что {a, b, c} ⊂ {a, b, c}? → Нет (не строгое, они равны)
