Тавтология — всегда истина

Тавтология — это утверждение, которое истинно при любых значениях входящих в него переменных. В математике тавтологии — это фундаментальные равенства, которые всегда верны.

1. Примеры тавтологий

  • x + 0 = x — нейтральный элемент сложения
  • x × 1 = x — нейтральный элемент умножения
  • x - x = 0 — вычитание самого себя
  • (a + b)² = a² + 2ab + b² — формула квадрата суммы

Пример: Использование тавтологии

Упростим: (x + 3)²

(x + 3)² = x² + 2·x·3 + 3²  
        = x² + 6x + 9

Это работает всегда — это тавтология.

2. Тавтология vs. Уравнение

  • Тавтология: всегда истинна (например, 2 + 2 = 4)
  • Уравнение: истинно только при определённых значениях (например, x + 2 = 4 → x = 2)

Пример: Не тавтология

x² = 4 — не тавтология, истинно только при x = ±2.

3. Зачем нужны тавтологии?

  • Они позволяют упрощать выражения без потери смысла.
  • Используются в доказательствах как аксиомы.
  • Лежат в основе алгебраических преобразований.

Советы

  • Запоминайте основные тавтологии — они как инструменты.
  • Используйте их для проверки: если после упрощения получилось тавтология — всё верно.
  • Тавтология — это "вечная истина" в математике.
